# Dart 学习指南

## 学习路径

### 第一阶段：Dart 基础语法
1. 变量和数据类型
2. 函数和方法
3. 条件语句和循环
4. 类和对象
5. 异步编程

### 第二阶段：Flutter 基础
1. Widget 系统
2. 布局系统
3. 状态管理
4. 路由导航

### 第三阶段：实战项目
1. Todo App
2. 天气 App
3. 社交 App

## 项目结构

```
dart_learning/
├── 01_basics/          # Dart 基础语法
├── 02_functions/       # 函数和方法
├── 03_classes/         # 类和对象
├── 04_async/           # 异步编程
└── 05_flutter/         # Flutter 项目
```
